# Assets Folder

Place custom images for the Animal Emergency Rescue Platform here.

## Suggested images

- `hero.jpg` - Hero banner (animal rescue themed)
- `logo.png` - App logo
- `reporter.jpg` - Reporter section banner
- `volunteer.jpg` - Volunteer section banner

## Usage

Reference in code as: `/assets/your-image.jpg`

Or update `src/assets/images.js` to use local paths.
